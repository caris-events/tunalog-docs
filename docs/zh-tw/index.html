現在就下載體驗！

Tunalog 可以在許多的作業系統上執行，你也可以前往 GitHub 發布頁面檢視更多不同架構的版本（如：x86、x64 或 ARM）。

從 GitHub 下載最新版本的 Tunalog：
$ wget -c https://github.com/caris-events/tunalog/releases/latest/download/tunalog_linux_x64
執行已下載的主程式：
$ ./tunalog_linux_x64

（若指定 -p :8123 啟動參數可以變更連接埠）

開啟瀏覽器，前往 http://localhost:8080 即可開始使用。

部落格的基本功能
易用的 Markdown 編輯器（由 SimpleMDE 提供）。
基本可自行編輯的佈景主題功能。
易於搬家的特性
使用 SQLite 檔案型資料庫，隨時都能帶著走。
使用相對網址，不須擔心換網址導致連結失效。
輕巧且支援主流系統
支援主流 Linux、Windows 與 macOS（包含 ARM 架構）。
程式僅有 30 MiB 大小，執行時僅消耗 32 MiB 記憶體。
