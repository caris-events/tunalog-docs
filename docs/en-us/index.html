Run the downloaded main program:

Basic Blog Features
Easy Markdown editor (powered by SimpleMDE).
Basic customizable themes.
Easy to Move
Portable, Zero-configuration SQLite file database.
Uses relative URLs, so links remain valid if the domain changes.
Lightweight and Cross-Platform
Supports Linux, Windows, and macOS (including ARM).
Only 30 MiB in size and uses 32 MiB of memory while running.
