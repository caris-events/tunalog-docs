现在就下载体验！

Tunalog 可以在许多操作系统上运行，你也可以前往 GitHub 发布页面查看更多不同架构的版本（如：x86、x64 或 ARM）。

从 GitHub 下载最新版本的 Tunalog：
$ wget -c https://github.com/caris-events/tunalog/releases/latest/download/tunalog_linux_x64
运行已下载的主程序：
$ ./tunalog_linux_x64

（若指定 -p :8123 启动参数可以更改端口）

打开浏览器，前往 http://localhost:8080 即可开始使用。

博客的基本功能
易用的 Markdown 编辑器（由 SimpleMDE 提供）。
基本可自定义的主题功能。
易于迁移的特性
使用 SQLite 文件型数据库，随时都能带着走。
使用相对网址，不需担心更换网址导致链接失效。
轻巧且支持主流系统
支持主流 Linux、Windows 和 macOS（包括 ARM 架构）。
程序仅有 30 MiB 大小，运行时仅消耗 32 MiB 内存。
